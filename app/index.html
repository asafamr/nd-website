What is NDJS?

NDJS is an open source framework that helps you create beautiful installers

Features

User experience
Cool things come in cool packages -
make a good first impression Use web styling capabilities for responsive, engaging UI that maintain your product look and feel
Quality
Installers are on your critical path -
ensure quality NDJS allows full testing and debugging capabilities for your code. Use modern IDEs and scripting languages, well known and tested frameworks and packages from NPM and Bower
Open source
Full configurability
NDJS is completely open source allowing you to tweak and configure it to your own needs

Technology

NDJS bundles the excellent NWJS platform to create native desktop applications with web styling capabilities.
The UI is designed with modern web technologies: HTML5, CSS3, JavaScript and rendered on the webkit engine (like chrome).
Use packages from Bower to upgrade the end user experience.
One well-documented, modern scripting language: JavaScript (or alternatives like TypeScript or CoffeeScript) with complete out-of-the-box debugging and testing capabilities.
Under the hood, NWJS uses Node, the well-known server side JavaScript engine.
This allows you to reuse thousands of tested packages from the NPM package library.

Pros, Cons and Other solutions

Well known alternative installer frameworks such as NSIS and Felxera's InstallShield create robust installers with low overhead.
NDJS approach is different and aims to provide a more visual appealing solution by trading-off some size overhead.
In addition, NDJS tries to take advantage of the web eco-system: using NPM and Bower packages, IDEs and excellent debugging and testing capabilities originally made for web development.

Some solutions try to hide the installation process altogether: InstallOnce, Squirrel, Chocolatey, App-stores and image distribution solutions such as Docker.
Unfortunately, these require some prerequisites in the end user target machine: installed frameworks, modern OS, in-house skills etc...
Some clients wish to install products only on their own controlled environment making image distribution problematic.
Some installation processes require elevated privileges which limit the use of certain solutions.
NDJS allows you to implement complex installation logic to overcome these.
